# University Events Hub – Jira Agile Project

## Overview

University Events Hub is a simulated Agile software project created to demonstrate practical Business Analysis, Product Management, and Quality Assurance skills using Jira.

The platform enables university students to discover campus events, register online, receive reminders, check in using QR codes, and download attendance certificates.

The objective of this project was not software development itself, but to simulate how a real Agile team plans, organizes, tracks, and validates product requirements using Jira.

---

## Project Objectives

* Demonstrate Agile Scrum practices
* Organize product requirements into Epics and User Stories
* Manage the Product Backlog
* Plan work across multiple Sprints
* Track implementation tasks
* Create Test Cases
* Report and manage Bugs
* Maintain traceability between requirements, testing, and defects
